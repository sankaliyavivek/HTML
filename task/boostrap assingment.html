Q 5) What is the diffrence between Bootstrap 4 and Bootstrap 5 ?

==>

Bootstrap 4 and Bootstrap 5 is a small diffrence it can class diffrence

Bootstrap 4 has five levels: xs, sm, md, lg, and xl. Bootstrap 5 has six levels: xs, sm, md, lg, xl, and xxl

Bootstrap 4 includes jquery and all associated plug-ins.

Jquery has been deleted and replaced with vanilla JS and several functional plug-ins in Bootstrap 5.

Q 6) What is a Button Group,and what is the class for a basic Button Group ?

==>

Button Groups in Bootstrap is a class of name “btn-group” which is used to create a series of buttons in groups (without spaces) vertically or horizontally.

Syntax: This is the basic syntax of the button group class where each button has its own class of “btn”.

Bootstrap provides 4 button sizes which you can add directly to your buttons by adding an additional class of “btn-group-*” to your “btn-group” class. No need to add separate classes to each button.

All 4 sizes can be used as follows

Vertical Button Groups: Bootstrap also supports vertical button groups stacked in a vertical manner rather than horizontally. Use the class “btn-group-vertical” to create a vertical button group

Q 8) In Bootstrap 4,what is flexbox ?

==>

Flexbox The biggest difference between Bootstrap 3 and Bootstrap 4 is that Bootstrap 4 now uses flexbox, instead of floats, to handle the layout.

The Flexible Box Layout Module, makes it easier to design flexible responsive layout structure without using float or positioning.

Set the direction of flex items in a flex container with direction utilities. In most cases you can omit the horizontal class here as the browser default is row.

However, you may encounter situations where you needed to explicitly set this value

Use .flex-row to set a horizontal direction (the browser default), or .flex-row-reverse to start the horizontal direction from the opposite side.

Q 9) How can one create an alert in Bootstrap ?

==>

Bootstrap 5 provides an easy way to create predefined alert messages

EX :

Success! This alert box could indicate a successful or positive action.

Alerts are created with the .alert class, followed by one of the contextual classes .alert-success, .alert-info, .alert-warning, .alert-danger, .alert-primary, .alert-secondary, .alert-light or .alert-dark

==>

To create Bootstrap cards, you need to understand the card class and its customizations and layouts.

A basic Bootstrap card consists of a . card class with the . card-body class, allowing you to add text, images, links, and other Bootstrap elements within a rounded border.

After researching and talking to professional developers, like Hubspot's Athena Ozanich, I've gained valuable insights about Bootstrap cards. As a student and mentor, I've come to appreciate the flexibility and consistency of Bootstrap cards. Let's now dive into the basics of working with Bootstrap cards.

Since cards are divs, they’ll span the entire width of the page by default. To set a width, you can apply CSS to your cards or place them in a grid, as we’ll see later.

You can also distinguish the body text of the card from the title using the .card-text class.
